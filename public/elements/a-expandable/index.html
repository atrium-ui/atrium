a-expandable

Implemented with Accordion, Combobox, Filter and Select.

Summary

  • A wrapper element, that can collapse and expand its content with an animation.
  • It can include a trigger elemeent which is always visible.

Props

  • opened (boolean: default = false)

    Controls the expand state of the element.

  • direction (“up” | “down”: default = “down”)

    Controls the direction the element will expand.

Slots

  • toggle

    The element that will trigger the expand state.
