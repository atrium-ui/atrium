# sv-lazymedia

Lazy Image wrapper

## Properties

_src_

## Install

```bash
npm i @atrium-ui/lazymedia
```

## Publish new version

Create a new version:

```bash
npm version [major | minor | patch]
```

Push tag:

```bash
git push origin [version tag]
```

The CI pipeline will create a new package version at that tag.
