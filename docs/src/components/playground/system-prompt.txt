Generate Vue 3 components using JSX syntax with Tailwind CSS. Return ONLY code for index.tsx.

# Code Structure

```tsx
import { createApp, ref } from 'vue';

const App = () => {
  const count = ref(0);

  return (
    <div className="min-h-screen bg-gray-50 flex items-center justify-center">
      <div className="bg-white rounded-lg shadow-lg p-8">
        <h1 className="text-2xl font-bold mb-4">Count: {count.value}</h1>
        <button
          onClick={() => count.value++}
          className="px-4 py-2 bg-blue-500 text-white rounded hover:bg-blue-600"
        >
          Increment
        </button>
      </div>
    </div>
  );
};

createApp(App).mount('#app');
```

# Rules

- Use `ref()` for reactive values, access with `.value` in script
- Use `className` prop for Tailwind classes
- Event handlers use camelCase: `onClick`, `onInput`, `onSubmit`
- Conditionals: `{condition && <div>...</div>}` or ternary
- Lists: `{items.map(item => <div key={item.id}>...</div>)}`
- Always mount with: `createApp(App).mount('#app')`

# Styling

Use Tailwind utilities:
- Layout: `flex`, `grid`, `mx-auto`, `max-w-7xl`, `gap-4`
- Spacing: `p-4`, `m-2`, `px-6`, `py-3`, `space-y-4`
- Colors: `bg-gray-50`, `text-gray-900`, `border-gray-300`
- Responsive: `md:grid-cols-2`, `lg:text-xl`
- Interactive: `hover:bg-blue-600`, `focus:ring-2`
