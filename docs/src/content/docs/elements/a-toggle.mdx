## Summary

- A functional helper element that sets an selected attribute on one or mutliple children.

## Props

- `multiple` (boolean: default = false)

  Controls if multiple children can be selected.

- `activeAttribute` (string: default = "selected")

  The attribute that the active child should have.

## Examples

```tsx
<a-toggle>
  <button>Item 1</button>
  <button>Item 2</button>
  <button>Item 3</button>
  <button>Item 4</button>
</a-toggle>
```
