## Props

- `enabled` (boolean: default = false)

  Whether the blur is enabled or not.

- `scrollLock` (boolean: default = true)

  Whether the blur should lock scrolling when shown.

## Examples

```tsx
<a-blur enabled>
  <div>
    <h2>Content</h2>
  </div>
</a-blur>
```
