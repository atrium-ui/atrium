## Configure package registry

Configure the package registry to install packages from the Gitlab package registry. More information about all published Packages can be found [here](https://gitlab.s-v.de/svp/atrium/-/packages).

<br/>

Update or create .npmrc file located next to the package.json file with the following content:

<Tabs>
  <TabItem label="npm">
    ```ini
    # .npmrc
    @sv:registry=https://gitlab.s-v.de/api/v4/projects/1560/packages/npm/
    ```
  </TabItem>
  <TabItem label="bun">
    ```toml
    # bunfig.toml
    [install.scopes]
    sv = "https://gitlab.s-v.de/api/v4/projects/1560/packages/npm/"
    ```
  </TabItem>
</Tabs>
