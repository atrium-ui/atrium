## Configure package registry

Configure your package manager to install packages from the Gitlab package registry. More information about all published Packages can be found [here](https://gitlab.s-v.de/svp/atrium/-/packages).

Update or create .npmrc file located next to the package.json file with the following content:

<Tabs>
  <TabItem label="pnpm" icon="pnpm">
    ```sh
    echo @sv:registry=https://gitlab.s-v.de/api/v4/projects/1560/packages/npm/ >> .npmrc
    ```
  </TabItem>
  <TabItem label="npm" icon="seti:npm">
    ```sh
    echo @sv:registry=https://gitlab.s-v.de/api/v4/projects/1560/packages/npm/ >> .npmrc
    ```
  </TabItem>
  <TabItem label="bun" icon="bun">
    ```toml
    # bunfig.toml
    [install.scopes]
    sv = "https://gitlab.s-v.de/api/v4/projects/1560/packages/npm/"
    ```
  </TabItem>
</Tabs>
