## Configure package registry

Configure your package manager to install packages from the Gitlab package registry. More information about all published Packages can be found [here](https://gitlab.s-v.de/svp/atrium/-/packages).

Create a .npmrc file in the root of your package to configure the registry:

<FileTree>
- src/
- README.md
- package.json
- .npmrc â¬…
</FileTree>

```sh
echo @atrium-ui:registry=https://gitlab.s-v.de/api/v4/projects/1560/packages/npm/ >> .npmrc
```
