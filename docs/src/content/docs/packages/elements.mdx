## Installation

Make sure your project is set up with access to the registry as described in [Setup](/sv-frontend-library/mono/setup).

<Command>
  <div slot="pnpm">
    ```sh
    pnpm add @svp/elements --save-exact
    ```
  </div>
  <div slot="npm">
    ```sh
    npm i @svp/elements --save-exact
    ```
  </div>
  <div slot="bun">
    ```sh
    bun add @svp/elements --exact
    ```
  </div>
</Command>
