## Installation

Make sure your project is set up with access to the registry as described in [Setup](/sv-frontend-library/mono/setup).

<Command>
  <div slot="pnpm">
    ```sh
    pnpm add @sv/elements --save-exact
    ```
  </div>
  <div slot="npm">
    ```sh
    npm i @sv/elements --save-exact
    ```
  </div>
  <div slot="bun">
    ```sh
    bun add @sv/elements --exact
    ```
  </div>
</Command>
