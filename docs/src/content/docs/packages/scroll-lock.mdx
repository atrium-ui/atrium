A small library for consitent scroll-locking across browsers.

Used in [a-blur](/atrium/elements/a-blur/).

## Installation

Make sure your project is set up with access to the registry as described in [Setup](/atrium/setup).

<Command>
  <div slot="pnpm">
    ```sh
    pnpm add @svp/scroll-lock
    ```
  </div>
  <div slot="npm">
    ```sh
    npm i @svp/scroll-lock
    ```
  </div>
  <div slot="bun">
    ```sh
    bun add @svp/scroll-lock
    ```
  </div>
</Command>

## Usage

```ts
import { ScrollLock } from '@svp/scroll-lock';

const scrollLock = new ScrollLock();

// lock page scrolling
scrollLock.enable();
// unlock page scrolling
scrollLock.disable();
```
