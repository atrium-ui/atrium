## Installation

Make sure your project is set up with access to the registry as described in [Setup](/sv-frontend-library/mono/setup).

<Command>
  <div slot="pnpm">
    ```sh
    pnpm add @sv/scroll-lock
    ```
  </div>
  <div slot="npm">
    ```sh
    npm i @sv/scroll-lock
    ```
  </div>
  <div slot="bun">
    ```sh
    bun add @sv/scroll-lock
    ```
  </div>
</Command>

## Usage

```ts
import { ScrollLock } from '@sv/scroll-lock';

const scrollLock = new ScrollLock();

// lock page scrolling
scrollLock.enable();
// unlock page scrolling
scrollLock.disable();
```
