## Installation

Make sure your project is set up with access to the registry as described in [Usage](/sv-frontend-library/mono/usage#configure-package-registry).

<Command>
  <div slot="pnpm">
    ```sh
    pnpm add @sv/codestyle
    ```
  </div>
  <div slot="npm">
    ```sh
    npm i @sv/codestyle
    ```
  </div>
  <div slot="bun">
    ```sh
    bun add @sv/codestyle
    ```
  </div>
</Command>
