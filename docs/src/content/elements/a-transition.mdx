#### Define custom transition animations using CSS:

[MDN](https://developer.mozilla.org/en-US/docs/Web/API/View_Transitions_API/Using#the_view_transition_pseudo-element_tree)

```css
@keyframes move-out {
  to {
    transform: translateX(-100px);
    opacity: 0;
  }
}

@keyframes move-in {
  from {
    transform: translateX(200px);
    opacity: 0;
  }
}

::view-transition-old[<name>] {
  animation: 0.33s ease both move-out;
}
::view-transition-new[<name>] {
  animation: 0.33s ease both move-in;
}

::view-transition-old(root),
::view-transition-new(root) {
  animation: none;
}
```
