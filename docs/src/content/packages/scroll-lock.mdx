A small library for consitent scroll-locking across browsers.

Used in [a-blur](/elements/a-blur/).

## Installation

Make sure your project is set up with access to the registry as described in [Setup](/guides/setup).

<Command
  pnpm="pnpm add @sv/scroll-lock"
  npm="npm i @sv/scroll-lock"
  bun="bun add @sv/scroll-lock"
/>

## Usage

```ts
import { ScrollLock } from '@sv/scroll-lock';

const scrollLock = new ScrollLock();

// lock page scrolling
scrollLock.enable();
// unlock page scrolling
scrollLock.disable();
```
